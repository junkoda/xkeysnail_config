# xkeysnail_config
Mac-like key bindings for Ubuntu using xkeysnail

xkeysnail:
https://github.com/mooz/xkeysnail

Memorandum:
https://qiita.com/junkoda/items/fc414a11f418766004f0 (Japanese)

Location of these files: `~/.xkeysnail`

Install xkeysnail globally:

```
$ sudo pip3 install xkeysnail
```

Without `sudo`, install could be local (~/.local/bin/xkeysnail)
